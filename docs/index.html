Welcome to SimpleBEM!

An unobstruvise CSS framework

SimpleBEM is a CSS framework focused on providing a lot of useful and recurrent styles for your application while not getting in your way. It accomplishes this by NOT setting any default styles to any component.

This is crucial, as this makes SimpleBEM completely optional: If you don't use any classes from SimpleBEM, you are not getting any styles out of it.
